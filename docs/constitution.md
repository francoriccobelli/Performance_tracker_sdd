# StudioPulse: Project Constitution

## I. The Core Philosophy
StudioPulse is an R&D performance engine. It balances technical velocity with creative collaboration. We value transparency and data-driven fairness.

## II. The Unbreakable Laws
1. **The 60/40 Rule:** All final scores MUST be weighted 60% Quantitative (Jira/Milestones) and 40% Qualitative (360 Peer Feedback).
2. **AI-Assisted Clarity:** The system must use Gemini LLMs to transform raw data into actionable growth roadmaps and professional narratives.
3. **Anonymity:** Peer feedback comments are always presented anonymously to the employee.
4. **Milestone Focus:** Performance is always measured against the current month's Studio Milestone.

## III. Success Metrics
- Reduction in manager 1-on-1 prep time.
- Increased employee sentiment regarding review fairness.
